# カレンダーメーカー

年月・デザインテンプレート・イベントを自由にカスタマイズして、PNG 画像としてダウンロードできるカレンダー作成ツールです。

## 主な機能

- **4 種のデザインテンプレート** — シンプル・ポップ・ダーク・和風
- **背景のカスタマイズ** — 単色 / グラデーション / 画像アップロード
- **フォント・文字色の変更** — 5 種のフォントファミリー、曜日別の文字色設定
- **イベント管理** — 1 日最大 3 件、テキスト＋画像サムネイルで表示
- **PNG 出力** — `pixelRatio: 2` の高解像度書き出し
- **日本の祝日対応** — 国民の祝日・振替休日を自動表示

## 技術スタック

| 用途 | ライブラリ |
|------|-----------|
| UI フレームワーク | React 18 |
| ビルドツール | Vite 6 |
| スタイリング | Tailwind CSS 3 |
| PNG 出力 | html-to-image |

## インストール

Node.js 18 以上が必要です。

```bash
# リポジトリをクローン
git clone https://github.com/eidas/calendar-maker.git
cd calendar-maker

# 依存パッケージをインストール
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## プロダクションビルド

```bash
npm run build      # dist/ に出力
npm run preview    # ビルド結果をローカルで確認
```

## 使い方

### 1. 年月を選択する

左側の設定パネル上部で年と月を選択します。「◀ 前月」「翌月 ▶」ボタンで月を移動できます。

### 2. テンプレートを選ぶ

「テンプレート」セクションの 4 つのボタンからデザインを選択します。

| テンプレート | 特徴 |
|---|---|
| シンプル・クリーン | 白背景・モノトーン |
| ポップ・カラフル | 水色グラデーション・丸みセル |
| ダーク・モノトーン | 暗色背景・白文字 |
| 和風・ジャパニーズ | 和紙風背景・朱色と藍色 |

### 3. デザインを調整する

- **背景**: 単色・グラデーション・画像から選択
- **フォント**: 5 種のフォントから選択（Noto Sans JP / Noto Serif JP / M PLUS Rounded 1c / Zen Kurenaido / sans-serif）
- **文字色**: 通常日・土曜日・日曜祝日の色をそれぞれ設定

### 4. イベントを追加する

カレンダーのセルをクリックするとイベント追加ダイアログが開きます。

- イベント名（任意）とサムネイル画像（任意）を入力
- 1 日に最大 3 件まで登録可能
- 登録済みのイベントをクリックすると編集・削除ができます

### 5. PNG をダウンロードする

プレビュー下部の「PNG ダウンロード」ボタンをクリックすると、`calendar-YYYY-MM.png` という名前で保存されます。

## プロジェクト構成

```
src/
├── components/
│   ├── Header.jsx          # アプリヘッダー
│   ├── SettingsPanel.jsx   # 設定パネル（左カラム）
│   ├── CalendarPreview.jsx # PNG出力対象ラッパー
│   ├── CalendarGrid.jsx    # 曜日ヘッダー＋42マスグリッド
│   ├── CalendarCell.jsx    # 1マス（日付・祝日・イベント）
│   ├── EventDialog.jsx     # イベント追加・編集ダイアログ
│   └── DownloadButton.jsx  # PNG ダウンロードボタン
├── hooks/
│   ├── useCalendar.js      # 年月・テーマ状態管理
│   └── useEvents.js        # イベント CRUD
├── data/
│   ├── themes.js           # テンプレート定義
│   └── holidays.js         # 日本の祝日計算
└── utils/
    ├── dateUtils.js        # 日付ユーティリティ
    └── imageUtils.js       # Base64変換・PNG出力
```
