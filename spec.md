# カレンダー画像作成アプリ — 設計仕様書

## 概要

ブラウザ上で動作する React アプリケーション。  
月単位のカレンダーを自由にカスタマイズし、PNG 画像としてダウンロードできる。

---

## 技術スタック

| 項目 | 選定 |
|------|------|
| フレームワーク | React（JSX 単一ファイル） |
| スタイリング | Tailwind CSS |
| PNG 出力 | html-to-image / html2canvas |
| 実行環境 | ブラウザ上で完結（サーバー不要） |

---

## 機能一覧

### 1. カレンダー表示

- **表示形式**: 月表示（1ヶ月ごと）
- **年月選択**: 任意の年・月を自由に切り替え可能
- **曜日ヘッダー**: 日〜土（日曜始まり）
- **前月・翌月の日付**: 薄いグレーで表示し、当月と区別

### 2. デザインテンプレート（4種）

ワンクリックでカレンダー全体のテーマを切り替える。

| テンプレート名 | 特徴 |
|---------------|------|
| シンプル・クリーン | 白背景、細いグリッド線、モノクロ基調 |
| ポップ・カラフル | 鮮やかな配色、丸みのあるセル、楽しい雰囲気 |
| ダーク・モノトーン | 暗い背景、白・グレーの文字、シックな印象 |
| 和風・ジャパニーズ | 和カラー（朱・藍・抹茶）、縦書き風ヘッダー |

### 3. カスタマイズ

テンプレートをベースに、以下の項目を個別に調整可能。

- **背景色 / グラデーション**: カラーピッカーで自由に設定
- **背景画像**: ユーザーがアップロードした画像を背景に配置
- **フォント**: 複数のフォントから選択（ゴシック系・明朝系・手書き風など）
- **文字色**: 通常日・土曜・日曜/祝日それぞれ個別に設定可能

### 4. 祝日表示

- **日本の祝日をプリセット**: 元日、成人の日、建国記念の日、天皇誕生日、春分の日、昭和の日、憲法記念日、みどりの日、こどもの日、海の日、山の日、敬老の日、秋分の日、スポーツの日、文化の日、勤労感謝の日
- **表示方法**: 祝日名をセル内に小さく表示し、日付を赤色（または指定色）で強調
- **振替休日**: 自動計算で対応

### 5. イベント管理（カスタムイベント）

ユーザーが任意の日付にイベントを追加できる。

#### 登録内容

| 項目 | 詳細 |
|------|------|
| 日付 | カレンダーのセルをクリックして指定 |
| イベント名 | テキスト入力（自由記述） |
| 画像 | ユーザーが画像ファイルをアップロード（任意） |

#### 表示仕様

- セル内に小さくイベント名を表示（テキスト）
- 画像がある場合はサムネイルとしてセル内に表示
- **1日あたり最大3件**まで登録可能
- テキストのみ、画像のみ、テキスト＋画像の組み合わせいずれも対応

#### 操作

- セルクリック → イベント追加ダイアログ表示
- 既存イベントクリック → 編集 / 削除

### 6. リアルタイムプレビュー

- テンプレート切り替え、カスタマイズ変更、イベント追加/編集のすべてが即座にプレビューに反映
- 編集パネルとプレビューを左右 or 上下に並べて表示

### 7. PNG 出力

- 「ダウンロード」ボタンでカレンダー部分を PNG 画像として保存
- プレビュー表示と同じ見た目で出力

## データ構造

### イベント

```typescript
type CalendarEvent = {
  id: string;
  date: string;        // "YYYY-MM-DD"
  title: string;       // イベント名
  image?: string;      // Base64エンコード画像（任意）
};
```

type ThemeConfig = {

  gridColor: string;

## 日本の祝日データ

アプリ内に祝日計算ロジックを組み込み、指定年の祝日を自動生成する。  
春分の日・秋分の日は天文計算による近似値を使用。  
振替休日（祝日が日曜に当たる場合、翌月曜が休日）も自動対応。

---

## 制約・備考

- サーバー不要、クライアント完結
- 画像はブラウザのメモリ上で処理（Base64）
- フォントは Google Fonts またはシステムフォントを利用
- 対応ブラウザ: Chrome / Edge / Safari / Firefox（最新版）
